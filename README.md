## 👋 Hey, I'm Andrew

I'm a **Computer Science** student at **CUNY Queens College** (graduating May 2027) who sits at the intersection of software engineering and design. I'm Frontend Lead & Event Manager for [Code For All], where I also designed the club's full brand identity from scratch — colors, typography, mascot and all.

I care a lot about *how things feel*, not just how they work. Whether I'm building a hackathon platform, designing a compiler, or sketching UI concepts, there's always a design sensibility behind every decision.

## 🚀 Featured Projects

### 🏰 HackKnight 2026 — Hackathon Event Platform
> React + Tailwind + Express.js · Vercel / Railway

Frontend Lead on a full hackathon event platform with a custom **lane-packing algorithm** for overlapping schedule rendering, a responsive SVG hero with a hand-crafted castle, and a token-authenticated Express backend.

---

### 🎨 Art Fundamentals Tutor — Gemini Hackathon
> React · Three.js · Gemini Pro Vision

An AI-powered app that teaches **1, 2, and 3-point perspective drawing** with real-time feedback via Gemini Flash. Canvas heatmap overlays highlight drawing errors on the user's actual sketch.


---

### 🎵 Museic — Artwork × Music Matcher
> Node.js · Google OAuth 2.0 · YouTube Data API · Art Institute of Chicago API

No npm, no frameworks, callbacks only — a pure Node.js HTTP server that matches museum artwork to YouTube playlist songs using dual API integration and a custom caching layer.
